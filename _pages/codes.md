## SeismicPhaseStatistics

A collection of tools for analyzing and stabilizing seismic phase. The
repository implements circular-statistics based methods to characterize
phase variability in seismic data, quantify wavefield coherence, and
design phase-stabilization workflows for multichannel recordings.

## Seismic Processing using Madagascar

A tutorial for end-to-end seismic processing of a 2D line using
[Madagascar](https://ahay.org/wiki/Main_Page), an open-source software
package for geophysical data analysis. The workflow covers fetching
data, setting geometry, first-break mute, ground-roll attenuation,
surface-consistent amplitude correction, velocity analysis and NMO,
stack, and Stolt's migration.

[![Repo](https://img.shields.io/badge/code-GitHub-24292e?logo=github)](https://github.com/arohatgi29/Seismic-Processing-using-Madagascar)

## LocalSignalAttributes.jl

A registered Julia package for computing *local* signal attributes —
measures defined in the neighborhood of each sample rather than
instantaneously or globally — using regularized inversion with shaping
regularization. Useful for estimating local frequencies and local
similarity between datasets, with applications in seismic signal analysis.

[![Repo](https://img.shields.io/badge/code-GitHub-24292e?logo=github)](https://github.com/arohatgi29/LocalSignalAttributes.jl)

## Hackathon on Computational Reproducibility

Winner of the [JSG Geoscience Hackathon](https://www.jsg.utexas.edu/geoscience-hackathon/past-hackathons/)
on Computational Reproducibility (Jackson School of Geosciences,
UT Austin). With Team Surfers, We reproduced Raissi et al. (2019) on
Physics-Informed Neural Networks, benchmarked CPU vs. GPU implementations,
ran sensitivity analyses on network depth and width for the 1+1D Burgers'
equation, and extended the framework to solve the 1D and 2D wave equations
under Dirichlet and Neumann boundary conditions.

[![Repo](https://img.shields.io/badge/code-GitHub-24292e?logo=github)](https://github.com/arohatgi29/Team_Surfers_Geoscience_Hackathon2024)
